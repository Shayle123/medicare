# MediCare+ Hospital Management Frontend

## Run steps

```bash
cd medicare-frontend
npm install
npm run dev
```

Open the local URL shown by Vite, usually http://localhost:5173/ or http://localhost:5174/.

## Demo login

Choose any role and use any email with password `123456`.
